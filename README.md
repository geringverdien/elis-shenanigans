# installation
- install a mod manager like [R2ModMan](https://github.com/ebkr/r2modmanPlus/releases) and create a profile for Straftat
- use the "import local mod" or similar feature to import the [.zip file in the "mod" folder](https://github.com/geringverdien/elis-shenanigans/raw/refs/heads/main/mod/mod.zip)
- launching the game via mod manager should now show the mod menu on launch
- press Insert to open/close the mod menu
